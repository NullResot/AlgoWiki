# AlgoWiki 详细使用手册

适用对象：普通用户、审核员、管理员、超级管理员。

本文只说明功能使用方式。部署、备份、故障处理请看 [维护手册](./维护手册.md)。

## 1. 角色能力

| 角色 | 主要能力 |
| --- | --- |
| 匿名用户 | 浏览公开内容 |
| 普通用户 | 个人中心、收藏、评论、提问、回答、修订、动态 |
| 学校用户 | 普通用户能力 + 指定范围内容审核 |
| 管理员 | 用户、内容、举报、审核、日志、站点配置管理 |
| 超级管理员 | 管理员能力 + 角色调整和高风险操作 |

## 2. 账号与个人中心

登录后进入个人中心，可查看：

- 用户 ID、角色、注册时间
- 昵称、性别、学校、简介
- 手机号验证状态
- 邮箱绑定状态
- 我发布的帖子
- 我发布的评论
- 我的收藏
- 我的问答和修订记录

个人中心默认展示资料摘要，不直接进入编辑表单。需要修改资料时，再进入对应子页面或编辑入口。

### 2.1 手机号验证

动态功能要求完成手机号验证：

1. 进入个人中心
2. 点击手机号验证按钮
3. 在弹窗中输入手机号并获取验证码
4. 填写验证码并提交

未验证用户访问动态页时，系统会提示先完成验证。

### 2.2 注销账户

用户可以申请注销账户。注销后：

- 账号不可继续登录
- 相关内容的作者展示为“已注销用户”
- 已发布内容按平台规则继续保留或由管理员处理

## 3. Wiki

Wiki 用于沉淀长期知识。

常见操作：

- 浏览分类和文章
- 搜索文章
- 收藏文章
- 评论文章
- 提交修订建议

修订建议进入审核队列后，由具备权限的用户处理。审核通过后才会更新正式内容。

## 4. 竞赛专区

竞赛专区包含：

- 赛事公告
- 赛程日历
- 补题链接
- Trick 技巧词条

Trick 列表采用分页展示，便于浏览和审核。提交内容应保持与算法竞赛相关，避免广告、灌水和无关内容。

## 5. 问答

问答用于短期问题讨论。

普通用户可以：

- 发布问题
- 编辑自己的问题
- 回答问题
- 采纳回答
- 关闭或重开自己的问题

管理员可以隐藏违规问题或回答。

## 6. 动态

动态功能类似轻量社区信息流。

普通用户可以：

- 发布文字动态
- 上传合规图片
- 评论动态
- 删除自己的动态或评论
- 举报违规动态或评论

管理员可以：

- 审核动态和评论
- 处理举报
- 手动精选或取消精选
- 删除动态或评论
- 在删除内容管理中查看和恢复内容

删除后的动态或评论不会再在普通动态页出现。只有管理员能在删除内容管理中查看归档。

## 7. 图片上传

图片上传会经过以下限制：

- 文件大小限制
- 图片数量限制
- MIME 类型与真实图片解码校验
- 图片重新编码和缩略图生成
- 图片安全审核
- 上传频率限制

图片审核未通过时，系统会提示原因或进入人工处理流程。

## 8. 审核台

审核台用于处理待审核内容，包含：

- Wiki 修订
- 补题链接
- 赛事公告
- 赛程
- 工单
- 文章评论
- 动态
- 动态评论
- 动态举报
- Trick
- 问题
- 回答

审核页会说明处理动作的影响。对于需要删除的违规内容，处理时应同步删除对应帖子或评论；帖子删除后，其评论同步从普通视图消失并进入删除内容存档。

## 9. 管理后台

管理后台包含：

- 用户管理
- 竞赛 Wiki 管理
- 竞赛专区管理
- 文档页管理
- 图片图库管理
- 动态管理
- 删除内容管理
- AI 助手管理
- AI 审核管理
- 访问统计
- 操作日志
- 安全日志

高风险操作应先确认对象、影响范围和恢复方式。

## 10. 使用规范

请不要发布：

- 违法违规内容
- 色情、暴力、血腥图片
- 人身攻击、骚扰、歧视内容
- 广告、引流和垃圾信息
- 正在进行中的比赛题目泄露或作弊内容
- 手机号、身份证、住址等个人敏感信息

违规内容可能被系统自动拒绝、进入人工审核或被管理员删除。
