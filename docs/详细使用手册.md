# AlgoWiki 详细使用手册

## 1. 手册适用对象

本手册面向以下人员：

- 普通使用者
- 学校/审核角色用户
- 管理员
- 超级管理员

如果你负责部署、巡检、备份或处理故障，请改看《维护手册》。

## 2. 系统访问方式

### 2.1 浏览器访问

部署完成后，直接通过域名或服务器公网 IP 访问首页。

示例：

- `https://example.com/`
- `http://<公网IP>:8001/`

### 2.2 登录与注册

登录/注册入口一般位于导航栏右上角。

用户注册后默认角色为普通用户。

注册时建议：

- 使用长期可用的邮箱
- 设置高强度密码
- 避免与其他网站共用密码

## 3. 角色说明

### 3.1 匿名用户

可以：

- 浏览首页
- 查看公开文章
- 查看公开问题与公告

不可以：

- 评论
- 收藏
- 提问
- 提交修订
- 进入个人中心

### 3.2 普通用户

可以：

- 编辑个人资料
- 修改密码
- 收藏文章
- 评论文章
- 发布问题和回答
- 提交文章修订建议
- 查看个人贡献历史和安全历史

### 3.3 学校用户

除普通用户能力外，还可以：

- 审核学校作用域下的内容
- 进入审核台处理部分修订

### 3.4 管理员

除前述能力外，还可以：

- 管理用户
- 发布公告
- 管理分类
- 治理文章、评论、问题、工单
- 查看操作日志与安全日志
- 查看后台概览

### 3.5 超级管理员

除管理员能力外，还可以：

- 调整关键用户角色
- 处理更高权限的用户治理操作

## 4. 首页与导航

首页通常包含：

- 站点简介
- 最新公告
- 热门或推荐内容
- 团队信息
- 导航入口

导航常见入口：

- 首页
- Wiki
- 问答
- 公告
- 扩展页面
- 个人中心
- 管理后台
- 审核台

## 5. 个人中心使用说明

## 5.1 查看个人资料

进入“个人中心”后，可查看：

- 用户名
- 角色
- 邮箱
- 学校名称
- 头像地址
- 简介
- 注册时间
- 最近贡献与操作记录

### 5.2 修改个人资料

可修改字段通常包括：

- 邮箱
- 学校名称
- 个人简介
- 头像 URL

注意事项：

- 邮箱必须唯一
- 建议头像使用稳定的 HTTPS 图片地址

### 5.3 修改密码

修改密码时需要：

1. 输入当前密码
2. 输入新密码
3. 再次确认新密码

系统会自动执行：

- 当前密码校验
- 新旧密码差异校验
- 最近密码复用校验
- Django 密码强度校验
- 修改成功后 Token 轮换

## 6. Wiki 使用说明

### 6.1 浏览分类

进入 Wiki 页面后，可按分类浏览文章。

常见操作：

- 通过左侧分类导航切换内容
- 搜索文章标题/摘要
- 查看全文

### 6.2 收藏文章

登录后可对文章进行收藏。

收藏后：

- 文章进入个人中心收藏列表
- 可在“我的收藏”中再次访问

### 6.3 评论文章

登录后可对文章发表评论。

支持：

- 新建评论
- 回复评论
- 删除自己的评论

请避免：

- 发布广告
- 恶意灌水
- 泄露个人敏感信息

### 6.4 提交修订建议

如果发现文章有误或需要补充：

1. 进入文章页面
2. 选择修订相关操作
3. 填写修订内容与理由
4. 提交审核

修订状态通常包括：

- `pending`：待审核
- `approved`：已通过
- `rejected`：已驳回

## 7. 问答模块使用说明

### 7.1 提问

进入问答页面后，可发布问题。

建议写清楚：

- 题目或问题标题
- 问题背景
- 已尝试的思路
- 需要帮助的点

### 7.2 回答

可以对问题发布回答。

建议回答包含：

- 解法思路
- 关键步骤
- 复杂度分析
- 必要时附示例

### 7.3 采纳回答

问题作者可采纳一个回答作为最佳答案。

### 7.4 草稿恢复

系统支持：

- 提问草稿
- 回答草稿

若误刷新页面，一般可以从本地草稿中恢复输入内容。

## 8. 公告与通知

### 8.1 公告

公告通常由管理员发布，用于：

- 系统通知
- 活动安排
- 维护通知

### 8.2 通知中心

登录用户可查看站内通知。

支持：

- 查看通知列表
- 查看未读数量
- 单条标记已读
- 全部标记已读

## 9. 审核台使用说明

面向学校用户、管理员、超级管理员。

常见功能：

- 查看待审核修订
- 按分类或状态筛选
- 批量通过/驳回
- 填写审核备注

操作建议：

- 对明显无效内容直接驳回并写明原因
- 对格式问题可驳回并给出修改建议
- 对高质量修订优先处理

## 10. 管理后台使用说明

## 10.1 管理概览

管理员可以查看：

- 用户统计
- 内容统计
- 近 7 天活跃趋势
- 事件分布

### 10.2 用户管理

可执行：

- 封禁
- 解封
- 软删除
- 恢复
- 批量治理
- 角色调整（超级管理员）

### 10.3 内容治理

可执行：

- 批量发布文章
- 批量隐藏文章
- 批量删除文章
- 批量隐藏评论
- 批量关闭/重开/隐藏问题

### 10.4 工单管理

可执行：

- 修改工单状态
- 批量修改状态
- 分派处理人
- 清空处理人

### 10.5 公告管理

管理员可创建、编辑、发布公告。

### 10.6 分类与扩展页面管理

管理员可维护：

- 分类显示/隐藏
- 分类顺序
- 扩展页面创建与编辑
- 比赛/友链/团队等扩展内容

## 11. 常见问题

### 11.1 登录后突然掉线

可能原因：

- Token 已过期
- 管理员封禁了账号
- 密码被修改后旧 Token 失效

处理方式：

- 重新登录
- 联系管理员确认账号状态

### 11.2 页面显示“加载失败”

可能原因：

- 后端服务不可用
- 数据库异常
- 服务器维护中

处理方式：

- 刷新页面重试
- 查看公告
- 联系管理员

### 11.3 上传图片失败

可能原因：

- 文件格式不支持
- 图片过大
- 服务器媒体目录不可写

建议：

- 使用 `png/jpg/webp/gif`
- 控制图片大小

## 12. 使用建议

- 不要在公开内容中泄露手机、身份证、家庭住址等敏感信息
- 密码不要与其他网站共用
- 管理员操作前先确认对象和范围
- 重要内容改动前建议先备份或留痕
