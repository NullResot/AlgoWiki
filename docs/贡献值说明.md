# AlgoWiki 贡献值说明

本文档面向 AlgoWiki 用户，说明贡献榜中的分数来源、各类榜单的计算方式，以及哪些行为会影响贡献值。

贡献值用于展示用户在 Trick、竞赛 Wiki、赛事资料维护和社区传播中的参与情况。贡献值不等同于账号权限、身份认证或官方背书。

## 总体计算

当前贡献榜分为四个独立方向：

| 方向 | 说明 |
| --- | --- |
| Trick 贡献值 | 来自 Trick 投稿、点赞、点踩与删除审核相关行为。 |
| Wiki 贡献值 | 来自竞赛 Wiki 修订通过审核后的有效修订。 |
| 赛事贡献值 | 来自锦标赛、赛事公告、补题链接等赛事资料维护。 |
| 社区贡献值 | 来自邀请真实新用户注册并完成验证。 |

贡献榜不会把不同性质的贡献混在一个总榜中。社区贡献值当前主要来自邀请系统。

## Trick 贡献榜

Trick 贡献榜展示用户在 Trick 内容建设和互动治理中的贡献。

当前规则：

| 行为 | 分值变化 | 说明 |
| --- | ---: | --- |
| Trick 通过审核 | +10 | 投稿通过审核后，作者获得贡献值。 |
| Trick 收到点赞 | +1 | 其他用户点赞后，Trick 作者获得贡献值。 |
| 取消点赞 | -1 | 点赞取消后，作者对应收益回滚。 |
| 发起点踩 | -1 | 点踩者消耗贡献值。 |
| Trick 收到点踩 | -2 | 被点踩的 Trick 作者扣除贡献值。 |
| 取消点踩 | 点踩者 +1，作者 +2 | 点踩取消后，双方对应变化回滚。 |
| 删除审核确认有效 | 点踩者 +2 | 管理员确认删除后，参与有效点踩的用户获得奖励。 |
| 已审核通过的 Trick 被管理员删除 | -10 | 如果删除的是已通过审核的 Trick，作者原投稿奖励会回滚。 |

部分 Trick 互动需要达到最低 Trick 贡献值：

| 功能 | 当前阈值 |
| --- | ---: |
| 点赞 Trick | Trick 贡献值达到 10 |
| 点踩 Trick | Trick 贡献值达到 10 |
| 触发删除审核 | 同一 Trick 累计 5 个点踩 |

## Wiki 贡献榜

Wiki 贡献榜展示用户对竞赛 Wiki 内容修订的贡献。

当前规则：

| 行为 | 分值变化 | 说明 |
| --- | ---: | --- |
| Wiki 修订通过审核 | 按有效新增或修改行数计算 | 10 行计 1 分，向上取整。 |
| 有效非空修订不足 10 行 | +1 | 只要存在有效新增或修改内容，最低计 1 分。 |
| 纯删除内容 | 0 | 纯删除不计贡献值。 |
| 相关文章被删除 | 回滚对应分值 | 只回滚曾经因该文章修订获得的 Wiki 贡献值。 |

有效行数只统计非空内容行。系统会对比修订前后的内容，只计算新增或修改后的有效内容行。

示例：

| 有效新增或修改行数 | 获得分数 |
| ---: | ---: |
| 1 行 | 1 |
| 10 行 | 1 |
| 11 行 | 2 |
| 20 行 | 2 |
| 21 行 | 3 |

## 赛事贡献榜

赛事贡献榜展示用户对赛事资料维护的贡献，包括锦标赛、赛事公告和补题链接。

当前规则：

| 行为 | 分值变化 | 说明 |
| --- | ---: | --- |
| 锦标赛通过审核 | +1 | 每条通过审核的锦标赛记录计 1 分。 |
| 赛事公告通过审核 | +5 | 每条通过审核的赛事公告计 5 分。 |
| 补题链接通过审核 | +2 | 每条通过审核的补题链接申请计 2 分。 |
| 对已有赛事公告的修订通过审核 | +5 | 奖励提交修订的人。 |
| 已计分的锦标赛、赛事公告或补题链接被删除 | 回滚对应分值 | 只回滚该条内容曾经产生的赛事贡献值。 |

赛事贡献榜适合用来查看谁在维护比赛时间、公告信息和赛后补题资源。

## 社区贡献榜

社区贡献榜当前主要展示邀请贡献。

当前规则：

| 行为 | 分值变化 | 说明 |
| --- | ---: | --- |
| 新用户使用邀请码注册 | 0 | 注册后先生成待生效记录，不立即加分。 |
| 被邀请用户完成手机号验证 | +1 | 邀请记录生效后，邀请人获得 1 点社区贡献值。 |
| 已生效邀请被回滚 | -1 | 对应邀请产生的分值会被扣回。 |
| 未生效邀请被驳回 | 0 | 未生效记录不产生分值，也不扣分。 |

邀请奖励的核心条件是：被邀请用户完成手机号验证。单纯注册账号不会直接给邀请人增加贡献值。

## 回滚与异常处理

以下情况可能导致贡献值回滚：

- 已通过审核的内容被管理员删除。
- 被邀请用户被封禁、注销或彻底删除。
- 被邀请用户的手机号验证被撤销。
- 管理员在邀请管理中手动回滚或驳回邀请记录。
- 管理员处理违规刷分、恶意互动或异常贡献记录。

系统会尽量只回滚对应行为产生的贡献值，不影响用户通过其他方式获得的贡献值。

## 不直接计分的行为

以下行为可能会记录为站内活动或审计记录，但当前版本不直接增加贡献榜分值：

- 浏览内容。
- 收藏文章或动态。
- 普通评论。
- 提交问题反馈。
- 发布动态。
- 管理员发布或管理站点公告。

后续如果这些行为纳入贡献值体系，AlgoWiki 会同步更新规则说明。

## 规则调整

贡献值规则会根据社区运行情况调整。后续如调整分值、阈值、统计范围或回滚规则，应同步更新本文档。
