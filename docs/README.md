# AlgoWiki 文档索引

本文档目录按用途拆分如下：

- 面向项目负责人和开发者的总体说明：[项目说明书](./项目说明书.md)
- 面向最终使用者和后台操作人员的功能说明：[详细使用手册](./详细使用手册.md)
- 面向运维和长期维护的部署、巡检、备份和故障处理说明：[维护手册](./维护手册.md)
- 面向首次远程部署、服务器初始化和旧版镜像 tar 流程的实操说明：[远程部署指南](./远程部署指南.md)
- 面向当前正式发布流程的一键更新说明：[发布与更新操作手册](./发布与更新操作手册.md)
- 面向 `develop/test/main` 三分支与 `test.algowiki.cn` 测试环境的说明：[分支与测试环境部署](./分支与测试环境部署.md)
- 面向 GHCR 方案背景和仓库侧配置说明：[GHCR部署与一键更新](./GHCR部署与一键更新.md)
- 面向 `www` 规范入口、HTTPS 源站与阿里云 CDN 保守接入方案的说明：[CDN与HTTPS上线蓝图](./CDN与HTTPS上线蓝图.md)
- 面向公开发布前检查的说明：[GitHub发布清单](./GitHub发布清单.md)

建议阅读顺序：

1. 首次接手项目：先看《项目说明书》
2. 准备培训使用者：看《详细使用手册》
3. 准备首次部署或迁移服务器：先看《维护手册》，再按《远程部署指南》执行
4. 准备日常发布、更新或回滚：直接看《发布与更新操作手册》
5. 准备使用内部测试站：看《分支与测试环境部署》
6. 准备把 `algowiki.cn` 正式切到 HTTPS + `www` 规范入口 + CDN：看《CDN与HTTPS上线蓝图》
