8. [维护手册](./维护手册.md)：看备份、巡检、日志和故障处理
9. [CDN 与 HTTPS 上线蓝图](./CDN与HTTPS上线蓝图.md)：看正式公网接入方案

## 文档规范

- 示例值一律用占位符表示，例如 `<SERVER_IP>`、`<REPO>`、`<STRONG_PASSWORD>`。
- 不在文档里写真实密钥、一次性验证码、Cookie、Token、数据库密码或个人隐私字段。
- 需要云控制台配置时，只保留“填什么、检查什么、验证什么”。
- 涉及上线流程时，优先写命令、检查点和回滚方式，少写叙述性说明。
