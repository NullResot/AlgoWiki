# AlgoWiki

AlgoWiki 是一个面向算法竞赛与知识协作场景的全栈平台，覆盖 Wiki、竞赛专区、动态、审核、管理与运维能力。

## 你可以在这里做什么

- 浏览和维护结构化知识库
- 提交 Wiki 修订建议和竞赛专区内容
- 发布动态、评论、举报与内容审核
- 管理公告、用户、站点内容和删除内容存档
- 使用手机号验证、图片审核、AI 审核和风控能力约束内容发布

> 问答模块当前默认关闭，不作为公开功能开放。需要恢复时由运维在对应环境中显式设置 `QA_MODULE_ENABLED=1`，并重新开放前端入口。

## 技术栈

- 后端：Django + Django REST Framework
- 前端：Vue 3 + Vite + Pinia + Vue Router
- 数据库：MySQL，兼容 SQLite 用于本地或应急场景
- 部署：Docker、Nginx、Gunicorn、GitHub Actions、阿里云 ACR / GHCR

## 目录概览

- `backend/`：Django 后端
- `frontend/`：Vue 前端
- `deploy/`：服务器部署模板、环境变量示例、运维脚本
- `scripts/`：本地构建、初始化、快部署脚本
- `docs/`：项目说明、使用手册、部署手册、运维手册
- `storage/`：日志、导出物、运行期文件
